# Lacan.js

> **Visualizing the Lacanian Subject in the Browser.**
> 在浏览器中重构拉康的主体拓扑学。

## About

**Lacan.js** 是一个探索性的 Web 可视化项目。雅克·拉康（Jacques Lacan）的精神分析理论高度依赖于结构与空间——从交叉的二维图式（Schemas）、层层叠加的欲望图谱（Graphs），到三维翻转的拓扑学（Topology）。

本项目旨在打破传统纯文本的阅读壁垒，利用现代 Web 图形技术，将这些静态的数学符号与精神分析结构转化为**可交互、可拆解、动态演变**的数字视觉。这不仅是一个图表库，更是一次用代码逻辑（Logic）去触碰无意识（Unconscious）的实验。

## Roadmap (实施路线)

本项目将逐步实现以下核心结构的动态交互：

- [ ] **Phase 1: Schemas (图式)** - L 图式 (Schema L): 象征轴与想象轴的交互连线
  - R 图式与 I 图式 (Schema R & I)
- [ ] **Phase 2: Graphs (图谱)**
  - 欲望图 (Graph of Desire): 从意指细胞到完整双层图的步进式生成
- [ ] **Phase 3: Topology (拓扑学)**
  - 莫比乌斯环 (Möbius strip) 的 3D 表面渲染
  - 波罗米奥结 (Borromean knot) 的物理引擎与拆解交互
